# I no longer use GitHub.

(For personal use, as of September 2023.)

My repos are being moved to https://git.aikuro.net/poems. I will probably leave
copies of at least my public repos here, but they will be archived.

I'm also mirroring a number of other people's repos to help decentralize the
source and/or act as a host incase the original is down for some reason:
https://git.aikuro.net/archives

Gitea is great, and there's no reason for me to use a proprietary git host
these days.

I will still be on GitHub for work, since my team uses it.
